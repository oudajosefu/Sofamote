# Roadmap

Sofamote is usable today, but the next launch-ready improvements are about trust, compatibility, and contributor runway.

## Near term

- Verify macOS and Linux release artifacts on real machines.
- Collect Windows install reports, especially firewall and lid-closed playback behavior.
- Add more streaming-site profiles with verified shortcut tables.
- Document common pairing, firewall, and Add to Home Screen troubleshooting.
- Add release checksums and verification instructions.

## Later

- Configure Windows signing and macOS notarization.
- Expand installer coverage beyond the current Windows, macOS, and Debian/Ubuntu artifacts.
- Improve profile auto-detection if it can be done without weakening the local-first security model.
- Add deeper automated checks for profile bindings and release packaging.

This roadmap is intentionally lightweight. If you want to help, start with issues labeled [`good first issue`](https://github.com/oudajosefu/sofamote/labels/good%20first%20issue) or [`help wanted`](https://github.com/oudajosefu/sofamote/labels/help%20wanted).
